# Coffee Shop Landing Page

This is a static landing page for a Manila coffee shop.

## Publish to GitHub Pages

1. Create a GitHub repository, for example `coffeeshop`.
2. Add a remote:
   ```bash
   git remote add origin https://github.com/<your-username>/coffeeshop.git
   ```
3. Push the `main` branch:
   ```bash
   git push -u origin main
   ```
4. Enable GitHub Pages in the repository settings:
   - In GitHub, go to **Settings > Pages**
   - Choose branch `main` and folder `/ (root)`
   - Save

Your site should then be available at `https://<your-username>.github.io/coffeeshop`.

## Notes

- This site uses `dist/output.css` generated by Tailwind.
- Do not commit `node_modules/`.
